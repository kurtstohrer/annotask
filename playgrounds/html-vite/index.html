About This Playground

This is a plain HTML playground for Annotask. No frameworks — just HTML, CSS, and a bit of vanilla JavaScript.

Annotask injects data-annotask-* attributes on elements during development, enabling the design tool to map visual changes back to source files.

  • Element inspection and selection
  • Live style editing
  • Source file mapping
  • Change report generation
